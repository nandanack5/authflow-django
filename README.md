# 🔐 AuthFlow

A secure authentication system built with **Django** and **Bootstrap 5**.

## ✨ Features

* User Registration
* User Login & Logout
* User Dashboard
* User Profile
* Edit Profile
* Change Password
* Flash Messages
* Responsive Navigation Bar
* Responsive Footer

## 🛠 Tech Stack

* Python 3
* Django
* Bootstrap 5
* SQLite
* HTML5
* CSS3
* Git & GitHub

## 📁 Project Structure

```
authflow/
│
├── accounts/
├── config/
├── static/
├── templates/
├── manage.py
└── requirements.txt
```

## 🚀 Installation

```bash
git clone <repository-url>
cd authflow

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```

Visit:

```
http://127.0.0.1:8000/
```
